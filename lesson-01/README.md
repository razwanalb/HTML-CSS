 Explanation:

<p> <!DOCTYPE html> → Declares this is an HTML5 document. <p>

<p> <html> ... </html> → Root element of the page. <p>

<p> <title> → Title shown in browser tab.<p>

<p> <body> ... </body> → Main visible content of the page. <p>

<p> <h1> → Heading. <p>

<p> <p> → Paragraph.<p> 

 HTML Tags
